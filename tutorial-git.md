# Tutorial GIT
autor: Hugo de Moraes

## Acessar seu repositório local
```
cd /Repositories/be-mean-instagram-mongodb
```
## Verificar o status
```
git status
```
## Adicionar alterações
```
git add .
```
## Commit
```
git commit -m 'Hugo de Moraes - MongoDb - Exercício "xx" - Resolvido'
```
## Subir
```
git push -u
```
